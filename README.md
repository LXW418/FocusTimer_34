# FocusTimer (极简番茄钟)

## 项目简介 (Introduction)
FocusTimer 是一个为大学生设计的轻量级命令行番茄工作法工具。它旨在帮助用户在期末复习期间保持专注，通过 25 分钟工作 + 5 分钟休息的循环，提高学习效率。

## 功能列表 (Features)
- [x] 基础倒计时功能
- [x] 自定义专注时长
- [ ] 专注数据统计 (开发中)
- [ ] 简单的提示音效

## 快速开始 (Getting Started)
1. 克隆项目:
   `git clone https://github.com/你的用户名/FocusTimer_XX.git`
2. 运行脚本:
   `python main.py`
